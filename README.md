## 🙋🏻 7. **What did I do**
-----

### - 구현 기능
- React, React-Native 개발 언어를 사용하여 식품 중개 플랫폼 서비스로 데모데이 출전
- 재사용 가능한 컴포넌트 폴더 구조 제안 후, 7일 시간 단축
- 회원가입, 자동 로그인, 리뷰 관리 기능 판매자, 관리자, 구매자 버전 별로 상태 관리 가능한 Redux와 Context 적용하여 개발

### - 구현 상세 설명
<details>
<summary>의뢰페이지</summary>
<div markdown="1">

- 의뢰폼 작성 및 등록
- 의뢰폼 수정 및 등록
- 의뢰 답변폼 작성 및 등록
- 의뢰 답변 채택 및 삭제 기능
- 의뢰 북마크 기능

</div>
</details>
<details>
<summary>사용기술 & 이슈</summary>
<div markdown="1">

- useRouter, Params 이용하여 작성자 유무 판단
- 답변 채택 유무 map 함수 순회하여 true 값이 하나라도 있다면 채택 옵션 제거
- 이슈:
의뢰 상세페이지 북마크부분이 컴포넌트 상태 끌어올리기가 안됐다.
- 해결:
컴포넌트 props로 주었던 값을 저장했던 setState 값을 같이 주어, 북마크 컴포넌트에서 값을 변경할 때 setState값을 변경하게 바꾸었다.

</div>
</details>
